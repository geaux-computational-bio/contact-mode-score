# BioDiff

Measure of protein-ligand complex (dis)similarity

## Installation

    $ cd src
    $ make biodiff -f makefile

## Usage

    $ biodiff --help

## Examples
    $ cd data
    $ sh biodiff.sh

## Dependencies

The codes have been tested on g++ 4.2.1, but any modern compilers supporting c++ standard libraries should work.
